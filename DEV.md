``````````````````````````````````````````# 🧠 Developer Notes — PhoneShop

Документ для быстрого понимания структуры проекта.

---

## 📦 PRODUCTS (apps/products)

### models.py
- Category — категории товаров
- Product — товары (name, price, image, category)

---

### views.py

#### product_list
- вывод всех товаров
- поиск через GET параметр `q`
- фильтр по категории `category`
- использует Q для поиска

#### product_detail
- страница одного товара

---

## 🛒 CART (apps/cart)

### cart.py
- хранит корзину в session
- методы:
  - add()
  - remove()
  - update()
  - get_items()

---

### views.py

#### cart_view
- отображение корзины

#### add_to_cart
- добавление товара

#### remove_from_cart
- удаление товара

#### update_cart
- изменение количества

---

## 👤 USERS (apps/users)

- регистрация
- логин / логаут
- использует встроенную auth систему Django

---

## 🎨 FRONTEND

### product_list.html
- карточки товаров
- поиск
- обработка пустого результата

---

### cart.html
- список товаров
- кнопки + / -
- итоговая сумма

---

### 404.html
- кастомная страница
- Matrix эффект (canvas)
- терминал анимация

---

## 🔍 ВАЖНЫЕ МОМЕНТЫ

### Поиск
- использует Q объекты для поиска по нескольким полям
- поиск по названию и категории
- ?q=iphone


---

### MEDIA
- хранится в /media/
- работает только при DEBUG=True

---

### Частые проблемы

❌ картинки не грузятся  
→ проверить MEDIA_URL и DEBUG

❌ корзина ломается  
→ session очистить

❌ поиск не работает  
→ проверить Q запрос

---

## 🚀 ИДЕИ ДЛЯ УЛУЧШЕНИЯ

- система заказов
- оплата
- избранное
- рейтинг товаров
- смена темы (как monkeytype)
- API (DRF)

---

## 💡 Быстрый вход в проект

1. models → структура
2. views → логика
3. templates → UI
4. cart → ключевая механика``````````````````````````````````````````
